The conceptual model

Do not ask whether it is “real” until you specify what kind of real.

The word real hides multiple independent questions. The book separates them so one answer cannot impersonate another.

History

Did this event happen to the system?

A memory can fail here. The represented childhood may never have occurred in the system’s causal history.

Historical authenticity
Function

Does the state reorganize the system?

The memory may guide attention, prediction, trust, avoidance, planning, and future learning.

Functional authenticity
Experience

Is there something it is like now?

This is the phenomenal question. It cannot be answered merely by checking whether the content is true.

Phenomenal authenticity
Identity

Is this the same subject over time?

Present consciousness, survival, continuity, copying, and personhood are related but not identical questions.
