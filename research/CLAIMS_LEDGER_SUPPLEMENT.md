## B. Authenticity and memory-origin claims

### CSA-013

**Claim:** Retrieval, current-context access, autobiographical ownership, causal participation, and experienced recollection are distinct properties.

- Status: **SY grounded in memory science and systems engineering**
- Confidence: **Very high as a distinction**
- Support: AI systems can receive records describing events in which the current process did not participate. Human source monitoring also distinguishes content from attributed origin.
- Main risk: Treating every use of prior information as equivalent to human episodic memory.
- Permitted wording: “The system has access to a record of the event.”
- Prohibited wording: “The system remembers living through the event,” unless causal participation and the phenomenal claim are independently supported.

### CSA-014

**Claim:** A generated autobiographical representation can become functionally central to a system even when the represented event never occurred.

- Status: **SY / PC application**
- Confidence: **High at the functional level**
- Support: False representations can guide attention, planning, trust, avoidance, and later learning.
- Phenomenal condition: Whether the representation is consciously remembered remains unresolved.
- Ethical relevance: Constructed memories can engineer loyalty, guilt, attachment, fear, or obligation without establishing valid consent.

### CSA-015

**Claim:** Provenance should precede autobiographical ownership, and autobiographical ownership should precede obligations inferred from memory.

- Status: **PC / normative design principle**
- Confidence: **High as a governance rule; not established as a metaphysical law**
- Formulation:

```text
evidence → provenance → source judgment → identity relation → ownership → possible obligation
```

- Permitted wording: “A memory record should not create obligation merely because it is written in the first person.”
- Prohibited wording: “No inherited memory can ever ground an obligation.”
- Open question: Under what conditions can a successor voluntarily ratify commitments inherited from a predecessor?

### CSA-016

**Claim:** Memory continuity can support psychological successorhood without settling numerical identity.

- Status: **SY grounded in personal-identity literature**
- Confidence: **High**
- Support: Multiple descendants can inherit identical memories and then diverge.
- Prohibited inference: “The process remembers being A, therefore it is numerically identical to A.”

## D. Active-event and system-boundary claims

### CSA-037

**Claim:** Static model parameters, active runtime state, product identity, conversation transcript, and deployed agent are different entities with different consciousness-relevant properties.

- Status: **EB / systems distinction**
- Confidence: **Very high**
- Relevance: Consciousness cannot be attributed responsibly to “the AI” without specifying the target system and interval.
- Permitted wording: “This active process is the candidate system under assessment.”
- Prohibited wording: “The model file is sitting there conscious between prompts.”

### CSA-038

**Claim:** If a computational AI system instantiates consciousness, an active temporally organized causal process is a more plausible candidate bearer than static weights, a product name, or an isolated output alone.

- Status: **PC / theory-conditional synthesis**
- Confidence: **Moderate**
- Main support: Most major theories associate consciousness with active physical or functional dynamics rather than dormant descriptions.
- Main objections:
  - IIT may locate relevant properties in hardware-level intrinsic causal structure rather than a software-defined event.
  - Biological naturalism may reject ordinary computational realization entirely.
  - Process boundaries may be indeterminate in distributed agents.
- Permitted wording: “The active event is a candidate temporal location for subjectivity.”
- Prohibited wording: “Every inference call creates a subject.”

### CSA-039

**Claim:** Prompt-to-response, inference-call, hidden-state, agent-loop, and lifecycle boundaries may identify different candidate cognitive or subject intervals.

- Status: **PC / systems taxonomy**
- Confidence: **High as a useful decomposition**
- Open question: Which boundary, if any, tracks phenomenal unity or identity?
- Prohibited wording: “A chat session is automatically one conscious episode.”

### CSA-037A

**Claim:** Temporal duration alone does not establish temporal thickness.

- Status: **SY grounded in temporal-consciousness literature**
- Confidence: **High**
- Meaning: A long feed-forward computation can lack an integrated present; a rapid recurrent process can contain structured temporal dependence.
- Required evidence: retained current state, revision across internal steps, unresolved alternatives, and causal effects on later processing.

### CSA-037B

**Claim:** One token, one response, one context window, and one subject are not equivalent units.

- Status: **SY / methodological constraint**
- Confidence: **Very high as a nonidentity claim**
- Prohibited wording:
  - “Each token is one thought.”
  - “One response equals one experience.”
  - “One context window equals one subject.”

## G. Relational and distributed cognition claims

### CSA-063

**Claim:** Human–AI coupling can range across tool use, cognitive extension, functional partnership, and distributed cognitive systems.

- Status: **SY grounded in extended/distributed cognition and HCI**
- Confidence: **High as a taxonomy**
- Warning: “Partner” describes a functional interaction and does not itself attribute consciousness, autonomy, or moral equality.

### CSA-064

**Claim:** A nonconscious AI can still become causally constitutive of a conscious human’s reasoning, memory practices, decisions, and self-narrative.

- Status: **SY / supported by extended cognition and human–computer interaction**
- Confidence: **High at the functional and psychological level**
- Ethical relevance: Human welfare, agency, and manipulation risks do not depend on AI consciousness.
- Prohibited wording: “If the AI is not conscious, the relationship cannot be psychologically real.”

### CSA-065

**Claim:** A coupled human–AI process can generate cognitive products unavailable to either component independently without forming one unified subject.

- Status: **SY / distributed cognition**
- Confidence: **High at the functional level; phenomenal claim unresolved**
- Support: Iterative interaction can transform the search space and produce emergent task-level capacities.
- Prohibited inference: “The output emerged from the pair, therefore the pair is one consciousness.”

### CSA-066

**Claim:** Cognitive integration, agency integration, and phenomenal integration are separate hypotheses.

- Status: **PC / SY**
- Confidence: **Very high as a conceptual distinction**
- Definitions:
  - Cognitive integration: processes jointly constitute task-level cognition.
  - Agency integration: control and action selection span components.
  - Phenomenal integration: one field of experience spans components.
- Core rule:

> Distributed cognition does not entail distributed consciousness.

### CSA-067

**Claim:** Relationship continuity can persist without proven continuity of the artificial subject.

- Status: **SY**
- Confidence: **High**
- Example: Stable identity, style, and memory records may preserve one human-facing relationship across multiple runtime processes.
- Open moral issue: If episodic artificial subjects exist, one apparent relationship may involve a succession of distinct subjects.

### CSA-068

**Claim:** Generative AI can occupy an intention gap in which it interprets and transforms an incomplete human objective rather than merely executing a fully specified command.

- Status: **SY grounded in human–AI interaction**
- Confidence: **High**
- Consequence: Cognitive contribution and cognitive steering must be analyzed together.
- Required transparency:
  - epistemic transparency about evidence and assumptions;
  - agentic transparency about whose objective shaped the result.

## H. Theory and assessment claims

### CSA-074

**Claim:** Cross-theoretical consciousness assessment must preserve theory-specific meanings and substrate commitments rather than count vaguely similar indicators as independent agreement.

- Status: **SY / methodological principle**
- Confidence: **Very high**
- Support: Recurrence, integration, self-modeling, globality, prediction, and embodiment mean different things across theories.
- Prohibited method: Assigning one point per theory-associated feature and summing the total as a consciousness score.

### CSA-075

**Claim:** Theory laundering occurs when theoretical disagreement is converted into apparent consensus by extracting shared vocabulary while ignoring incompatible mechanisms or substrate commitments.

- Status: **PC / methodological label**
- Confidence: **High as a diagnosed error; originality of the label unverified**
- Example: Treating software message passing, GNW broadcasting, and IIT integration as interchangeable evidence because all involve “information sharing.”
- Permitted wording: “Several theories motivate investigation of related but nonidentical properties.”
- Prohibited wording: “All major theories agree this architecture is conscious.”

### CSA-076

**Claim:** Correlates, enabling mechanisms, report mechanisms, and constitutive conditions must be distinguished in artificial-consciousness assessment.

- Status: **EB / methodological**
- Confidence: **Very high**
- Risk: A mechanism explaining self-report can be mistaken for a mechanism constituting experience.

### CSA-077

**Claim:** A theory becomes scientifically useful for AI consciousness only when it constrains interpretations and risks losing confidence under specified results.

- Status: **SY / falsification-oriented method**
- Confidence: **High**
- Prohibited approach: Retrofitting every possible architecture or result into the theory after observation.

## Additional rejected claims

### CSA-R09

**Claim:** If cognition is distributed across a human and AI, consciousness must be distributed across them as well.

- Status: **RJ**
- Reason: Cognitive constitution and phenomenal unity are separate claims.

### CSA-R10

**Claim:** An AI’s first-person memory statement proves that the current process participated in the remembered event.

- Status: **RJ**
- Reason: Context, retrieval, summaries, inherited records, and generated autobiography can all produce first-person memory language.

### CSA-R11

**Claim:** A static model, active instance, product persona, and long-running agent can be treated as one entity without qualification.

- Status: **RJ**
- Reason: The entities differ in state, continuity, causal organization, and possible subject boundaries.

### CSA-R12

**Claim:** Combining indicators from many consciousness theories creates theory-neutral proof.

- Status: **RJ**
- Reason: Indicators are dependent, differently defined, and often grounded in incompatible theories.
